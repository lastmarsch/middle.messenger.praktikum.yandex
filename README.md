## Страница проекта

Docker-сборка проекта развёрнута на хостинге Heroku и доступна по [ссылке](https://yp-chat-lastmarsch.herokuapp.com/).

  
## Сборка и запуск
### Собрать проект
    npm run build
### Запустить проект на 3000 порту
    npm run start
### Запустить тесты
    npm run test
